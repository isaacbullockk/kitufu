# AFCON 2027 Uganda Booking Platform — Plan

## Overview
Build a Booking.com-like web platform for Africa Cup of Nations 2027 in Uganda.
Brand: "Pamoja Residences" (AFCON Hubs)
Key concept: Pop-up fan residences in converted/empty buildings, bundled with shuttles.

## Stage 1 — Design & Architecture
- Load `vibecoding-webapp-swarm` skill
- Design system: African-inspired warm palette, clean modern UI
- Core features:
  1. Hero — AFCON 2027 Uganda branding with search
  2. Search/Filter — Location (Kampala, Hoima), dates, guests
  3. Property Listings — Grid of available residences with pricing
  4. "Pamoja Residences" special category
  5. Property Detail — Amenities, shuttle info, bulk booking CTA
  6. Booking Flow — Date selection, guest count, shuttle add-on
  7. Group Booking — "Buy Out Building" feature for supporters' clubs
  8. Season Pass — Group stage duration booking
  9. Shuttle bundling — "Stay + Shuttle to Stadium"
  10. Dashboard — User bookings, host dashboard

## Stage 2 — Development
- React + TypeScript + Tailwind CSS + shadcn/ui
- Build all screens and components
- Responsive design

## Stage 3 — Deploy
- Deploy as static webapp
